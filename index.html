Checking if branching is working or not!
Yup, branching is working just fine! Points to be noted :-
1. Clone a Repository from GitHub
2. Create different branches to work on different branches.
3. Make changes to those branches and Commit them to current branch.
4. Create a Pull Request to merge the Current changes branch to the main branch on GitHub.
5. Merge Pull Request to main branch on GitHub and then Changes are Live in production.
6. Fetch the changes that have been pushed to main branch on GitHub on your local Main branch.
